# Daily Workflow Guide — 每日使用手册

> **文档定位**：系统上线后每天怎么用。从早上到晚上的标准流程 + 应急情境处理。  
> **目标读者**：3 个月后的你，系统已经跑起来，但可能在某个时刻陷入混乱

---

## 一、每日标准时间表

### 美东时间 08:30 — 起床后（15 分钟）

**要做的事**：
1. 打开 Telegram，看 **Regime 晨报**（系统自动推送）
2. 心里有数：今天是激进 / 标准 / 谨慎 / 不交易 哪一档
3. 看 watchlist top 5 异动

**关键数字**：
- Regime Score：今天的"天气预报"
- VIX：超过 25 额外小心
- 今日宏观事件：FOMC / CPI 等

**输出**：你知道今天能不能交易、可以追突破还是只能等 retest

---

### 09:00-09:30 — 准备开盘（30 分钟）

**如果 Regime Score ≥ 55**：
1. 打开项目的 Today 仪表盘
2. 过一遍 Top 5 异动的股，看**日线结构**（用 TradingView Widget）
3. 标注哪几只有 setup：
   - 价格站上 EMA13（日线）？
   - 昨日高点在哪里？
   - 盘前 VWAP 是多少？
4. 准备好你的**候选名单**（最多 3 只）

**如果 Regime Score < 55**：
1. **不准备任何交易**
2. 打开 Journal，看昨天的交易，复盘
3. 或者准备 Shadow Trade（练盘感）

**切记**：候选名单 ≠ 会交易。只是圈定"如果有 setup 我只看这些"。

---

### 09:30-10:00 — 开盘第一小时（硬禁令时段）

**硬规则：不下新单**。

**为什么**：
- 你历史数据显示这段时间 29% 的订单但胜率最低
- 开盘订单簿不平衡，"突破"往往是假的
- 算法 / 大户常在开盘利用流动性"扫单"

**可以做的事**：
- 观察大盘方向是否符合盘前预期
- 观察你候选名单的股：是真的强还是装强？
- 在 Shadow Trades 里记录"如果我现在买 X 会怎样"（学习用）

**绝对不能做**：
- 追开盘冲高的 0DTE Call
- "先买一点试试"
- "错过了会后悔"的 FOMO 心态

---

### 10:00-11:30 — 第一交易窗口（最佳时段）

**这是你真正可以交易的第一个时段**。

**按流程走**：
1. 候选股出现突破信号 → 打开 Breakout Status 面板
2. 系统检查 Q1-Q5 五道过滤
3. 全过 → 决定 Retest 等待 或 分批试仓
4. Retest 失败 → 笑一下，避过一次假突破
5. Retest 成功 → 入场，设 EMA13 止损

**心态关键词**：**耐心**。宁可错过，不要追高。

**频率限制**（硬规则）：
- 11:30 之前最多下 3 单
- 同一 underlying 最多 1 单
- 如果已经错过信号，不要因为"不甘心"强行入场

---

### 11:30-13:30 — 午盘（低流动性）

**默认不交易**。

**为什么**：
- 流动性最低，单笔大单能推出假信号
- 机构交易员在吃午餐，纯散户博弈
- 历史数据：这段时间你胜率尚可但回报低（因为波动小）

**例外**：
- 已有持仓正常管理
- 看到明显 retest 机会（层 2/3 全过）
- 有新闻驱动（财报 / 重大公告）

**手痒怎么办**：
- 关闭盘面，看 1 小时书 / 打盹
- 或者打开 LEAP Explorer 学点长期思维

---

### 13:30-15:00 — 第二交易窗口

**可以交易，但比早盘保守**。

**关键观察**：
- 大盘方向是延续还是反转？
- 你的持仓是否仍守在 EMA13 上方？
- 有没有新的信号出现？

**加仓规则**：
- 如果早盘的持仓还在赚钱且在 EMA13 上方 → 可以考虑加仓
- 但**不要超过原计划的 1.5 倍**
- 加仓后止损位必须上移

**新开仓规则**：
- 仍遵守 Breakout Filter
- 下午开仓更倾向"持仓过夜"的考虑（期权要算 theta）

---

### 15:00-16:00 — 尾盘（特殊时段）

**0DTE 期权最危险时段**。

**规则**：
- **15:30 后不开新 0DTE 仓**
- 0DTE 持仓要考虑平仓（不平可能 close 在行权价附近导致 exercise 风险）
- 其他 DTE 持仓正常管理

**尾盘拉升 / 砸盘**：
- 常见于期权结算日（周五）
- 不追
- 如果你的持仓被带走，就接受

---

### 16:00-17:00 — 盘后（复盘时间）

**必做的事**：
1. Moomoo 导出当日 CSV → 放 `~/Daily-Stock-Inbox/`
2. 等 30 秒，系统自动入库
3. 打开 Journal → 看 **Daily Health Check**

**检查清单**（5 分钟）：
- [ ] 今日订单数：< 10？（目标）
- [ ] 0DTE 占比：< 50%？
- [ ] 开盘 30 分钟内下单：0 笔？（硬目标）
- [ ] 单一标的最多 3 次？
- [ ] Regime Score 是否 < 55 但仍交易？

**如果 Health Check 显示违规**：
- **在 Journal 写一句话**：为什么违规？（不写就没效果）
- 注明情绪状态：FOMO / 报复 / 无聊 / 其他

---

### 19:00-20:00（可选）— 深度复盘

**每周 1-2 次做**：

1. 打开 Journal 选一笔今天最大盈或最大亏的交易
2. 看 AI 分析（系统自动生成）
3. 手动在 user_notes 补充"当时的心理状态"
4. 看该交易的 Regime Score + Breakout Filter 指标 → 回顾自己是否按系统

---

## 二、每周日晚（30 分钟）

### 周日 18:00-19:00

打开 Weekly Reality Test 报告：

**必看的 5 个数字**：
1. 本周胜率（对比过去 4 周均值）
2. Profit Factor（应该 ≥ 1.5）
3. 去掉 Top 3 盈利笔后是否仍盈利？
4. 开盘 30 分钟内下单占比（目标 < 10%）
5. Retest vs Chase 胜率差（差应该 > 15 个百分点）

**问自己 3 个问题**：
1. 本周是否至少有 2 天遵守"Regime < 55 不交易"？
2. 本周 Retest 次数是否多于 Chase 次数？
3. 本周最大亏损笔是否在 Regime < 55 的日子？（如果是，那是系统胜利）

**规划下周**：
- 如果本周表现好 → 保持，不要因为"感觉好"激进
- 如果本周表现差 → 降低仓位下周（每笔仓位 × 0.7）

## 三、每月 1 号（90 分钟）

### 早晨 09:00 之前

**读 Monthly AI Retrospective**（系统自动生成）。

报告会覆盖：
1. 数据快照
2. Reality Test（去掉 Top 5/10/10% 的盈亏）
3. 与上月对比（DTE 分布 / 时段 / 标的集中度）
4. 重复错误模式
5. Phase 目标进度
6. 下月 3 条可执行规则

### 读完后的 30 分钟

在 `Journey > Monthly Reflection` 手动补充：
- 上月最重要的一次"按系统"经历（写 2 句）
- 上月最典型的一次"违反系统"经历（写 2 句）
- 下月自己想改进的 1 件事（不超过 1 件，多了做不到）

### 每月最后一件事

如果 Phase 0 Month 2 结束：
- 跑 `SELECT ... FROM trades JOIN regime_scores ...` 看 regime 系统有效性
- 判断是否进 Phase 1

## 四、应急情境手册

### 情境 A：连续亏 3 笔

**立刻做的 4 件事**：
1. **停手**。关掉下单界面
2. 写一条备注：上 3 笔分别什么情况？是系统内还是系统外？
3. **如果 3 笔都是追突破** → 今天剩余时间只做 retest
4. **如果 3 笔都在 Regime < 55 的日子** → 今天不交易了

**绝对不能做**：
- 为了"扳回来"加大仓位
- 去做 0DTE 深度 OTM 赌大的
- 切换标的继续追

**道理**：连续亏损说明**策略 / 心态 / 环境**三者至少一个出错。停手是唯一理性决定。

---

### 情境 B：账户当日 -3%

**这是硬性冷静线**。

**立刻**：
1. 关闭所有 0-3 DTE 持仓（theta 会加速损失）
2. 其他持仓**不要平仓**（避免非理性）
3. 停止下任何新单
4. 去吃饭 / 散步 / 睡觉

**明天**：
- Journal 复盘：-3% 的具体来源
- 如果是"一笔大的"导致：检查仓位是否超过单笔 2% 风险
- 如果是"多笔叠加"：检查是否连续犯同一个错误

---

### 情境 C：系统给 Regime < 40 但你觉得"今天能做"

**这是最危险的情境**。

**问自己**：
- 你凭什么觉得能做？是具体数据还是"感觉"？
- 如果你能具体列出 3 条反对系统的数据依据 → 下小仓（1/3）试试，但要在 Journal 详细记录
- 如果只是"感觉" → **相信系统**，不交易

**心法**：
> 你过去 3 个月的"感觉"胜率 34%。系统的规则是数据驱动的。除非你有具体证据否定系统，否则相信它。

---

### 情境 D：错过了一个你看中的大行情

**症状**：TSLA 今天涨 5%，你早上看到但没按系统入场，结果错过。

**应对**：
1. **庆祝自己没追高**（如果追高了可能赔得更惨）
2. Journal 记录："错过 TSLA，按系统应该 XXX，我没做对的地方是 YYY"
3. **不要为了补偿进别的仓位**（最常见的错误）

**长期视角**：
- 每周都会有 3-5 只股大涨
- 你抓到其中 1-2 只就够了
- **"错过一次" 永远比 "踩中一次假突破" 更安全**

---

### 情境 E：系统故障 / 数据拉不到

**Telegram 没收到晨报**：
1. 手动跑 `python -m src.regime.cli --no-save`
2. 看是不是某个数据源挂了
3. 如果 yfinance 全挂 → 直接看 Bloomberg Terminal 或 TradingView 判断 Regime（手动）

**Journal CSV 没自动导入**：
1. 检查 `watchdog` 服务是否在跑
2. 手动跑 `python -m src.journal.cli import ~/Daily-Stock-Inbox/History-xxx.csv`
3. 查看日志 `~/daily_stock.log`

**AI 分析失败**：
1. 不阻塞入库
2. 手动在 UI 点"Re-analyze"

---

### 情境 F：收到重大突发新闻（Trump 推文 / Fed 讲话等）

**优先级处理**：

**Trump 发关税相关 Truth Social 贴**：
1. 立刻关闭所有**不是 0DTE 的多头仓位**（避免盘中突变）
2. 0DTE 持仓看情况：如果离到期 < 2 小时可以持有，否则平仓
3. **不立刻加空头仓位**（反应过激是散户典型错误，机构会先砸再拉）
4. 等 30-60 分钟观察市场反应再决定动作

**Fed 官员鹰派讲话**：
- 通常影响大盘 30-90 分钟
- 不做相反方向的抢反弹
- 已有持仓如果在 EMA13 附近，按系统止损

---

### 情境 G：系统发出 Breakout Setup 信号但我不在电脑前

**移动端应急**：
1. Telegram 会推送信号
2. 手机 Moomoo 查当前价
3. 只做"跟随系统建议"的动作，不因为"在外面不方便盯"而改仓位
4. 如果不确定，**不入场**

**原则**：移动端不是你的常规交易环境。移动端出错率是桌面端的 3 倍。

## 五、每日必看的"3 个数字"

如果你时间极紧，每天盘后只看这 3 个：

### 数字 1：Daily Health Check 里的"Warnings 数"
- 0 个 → 今天按系统了 ✅
- 1-2 个 → 小瑕疵，下次注意
- 3+ 个 → 脱轨了，认真复盘

### 数字 2：今日 PnL / (当日总仓位)
- > +2% → 好日子，别飘
- -2% - +2% → 常态
- < -2% → 警钟（情境 B 处理）

### 数字 3：0DTE 占比
- < 50% → 正在好转 ✅
- 50-80% → 老毛病
- 80%+ → Phase 0 Week 1 的你

## 六、每周必做的"3 件事"

### 周一开盘前
- 把 Weekly Reality Test 看完
- 定下本周"禁止清单"（至少 1 条新规）

### 周三中午
- 查看本周过半的数据：是否朝着目标走？
- 如果明显脱轨 → 周四周五降低仓位

### 周五盘后
- 导出本周 CSV
- 准备好周日晚的深度复盘

---

## 七、每月必做的"1 件事"

### 月度复盘会议（一个人）

找一个安静的时段（2 小时），打开 Monthly AI Retrospective，**把报告完整读完并回答所有问题**。

写一份"月度反思笔记"（100 字够）：
- 这个月我的主要改进是什么？
- 下个月我最想改变的 1 件事是什么？
- 对当前 Phase 我还有多远？

把这份笔记保存在 `Journey > Monthly Reflections` 里。**3 个月后回看会震惊于自己的变化**。

## 八、长期视角（12 个月后回看）

**如果这套系统真的起作用**，12 个月后你应该：

1. **日均订单**：从 15 → 5 以下
2. **胜率**：从 35% → 50%+
3. **Profit Factor**：从 1.36 → 2.0+
4. **0DTE 占比**：从 85% → 30% 以下
5. **LEAP / 正股占比**：从 0% → 50%+
6. **账户曲线**：从过山车 → 稳定上升
7. **心理状态**：从每天紧张 → 按规则执行不焦虑

**如果 12 个月后这些都没变化**：
- 不是系统没用
- 是你没执行系统
- Journal 数据会清晰显示你在哪个环节脱轨

**项目的最大价值**：**你骗不了自己**。数据一直在那里。

## 九、反复提醒的核心原则

1. **相信系统 > 相信盘感**
2. **错过 > 踩中假突破**
3. **Retest > Chase**
4. **Regime 决定能不能做，Breakout Filter 决定具体做哪个**
5. **Journal 的诚实 > 账户的漂亮数字**
6. **月度胜率上升 > 单笔暴利**
7. **Phase 进度 > 短期盈亏**

把这 7 条打印出来贴在显示器边上。
